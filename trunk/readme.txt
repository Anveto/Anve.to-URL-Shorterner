Automatically shortens outgoing links in your posts using Anve.to.

== Description ==

This plugin automatically shortens outgoing links in your posts using the Anve.to API which is free to use. Users can view statistics of links that have been shortened in the Anve.to dashboard.

Statistics include:

- Number of clicks
- Geolocation
- Link bundles
- Social network tracking
- Custom link alias
- Public/private link statistics
- Password protected statistics

Premium users

- Custom splash pages
- Export statistics

== Development ==

You will find the official Github repository here https://github.com/Anveto/Anve.to-URL-Shorterner

== Installation ==

1. Upload `anveto-urlshorterner` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Anveto settings page via the admin area and fill in your Anve.to API key. Links will then automatically be shortened and stored in your Anve.to account when you publish or edit a post.
